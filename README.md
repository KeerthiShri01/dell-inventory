# Dell Inventory Manager

A simple inventory management system for Dell products built using HTML, CSS, and JavaScript.

## Features
- View all products in the inventory
- Add new products
- Remove existing products
- Check if a product is in stock or out of stock

## Technologies Used
- HTML
- CSS
- JavaScript (Arrays, Functions, DOM Manipulation)

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
